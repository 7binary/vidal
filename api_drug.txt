API для получения аналогов препаратов.

Страница запроса: https://www.vidal.ru/api/drug/equal/{ProductID}/{Type}
Метод запроса: GET
Безопасность: Basic Auth. Логин: "rambler", пароль: "Cv1Fbuwl2EVxEaifriFrcZ3CpLrP03X"
Для безопасности дополнительно необходимо в каждый запрос добавлять заголовок X-Token

Описание параметров:
1. ProductID - идентификатор исходного препарата
2. Type - число от 0 до 4 включительно. Это тип поиска.
0 = Полные аналоги
1 = Близкие аналоги
2 = Приблизительные аналоги
3 = Препараты с близкими свойствами
4 = Все категории

Пример запроса: https://apidrug:equal@www.vidal.ru/api/drug/equal/3782/4
Пример ответа в формате JSON:
{
    "a0": [
        29959,
        6701,
        11186,
        1659,
        6624,
        30115
    ],
    "a2": [
        6702,
        31720,
        19090
    ]
}

Нашлись полные аналоги, они сгруппированы в "a0".
Нашлись приблизительные аналоги, они сгруппированы в "a2".
Все препараты уже упорядочены по группам и названиям препаратов в алфавитном порядке.
